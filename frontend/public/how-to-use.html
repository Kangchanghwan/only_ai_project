🚀 빠른 시작 가이드

1룸 생성하기

메인 페이지에서 "새 룸 만들기" 버튼을 클릭하면 자동으로 룸 코드가 생성됩니다.

  • 룸 코드는 100000 ~ 999999 사이의 6자리 숫자입니다
  • 생성된 룸 코드를 다른 사람과 공유하세요
  • 같은 룸 코드를 입력한 모든 사용자가 파일을 공유할 수 있습니다

2룸 입장하기

이미 생성된 룸에 참여하려면:

  • 메인 페이지에서 6자리 룸 코드를 입력하세요
  • "입장" 버튼을 클릭하면 즉시 룸에 참여됩니다
  • 또는 QR 코드를 스캔하여 빠르게 입장할 수 있습니다

3파일 업로드하기

룸 안에서 파일을 공유하는 방법:

  • 방법 1: "파일 선택" 버튼을 클릭하여 파일 선택
  • 방법 2: 파일을 드래그 앤 드롭으로 업로드 영역에 놓기
  • 방법 3: 클립보드에서 직접 붙여넣기 (Ctrl+V / Cmd+V)
  • 업로드된 파일은 같은 룸에 있는 모든 사용자에게 실시간으로 표시됩니다

4파일 다운로드하기

공유된 파일을 받는 방법:

  • 파일 카드의 "다운로드" 버튼을 클릭하여 개별 다운로드
  • 여러 파일을 선택한 후 "선택한 파일 다운로드" 클릭
  • 모든 파일을 ZIP으로 압축하여 한 번에 다운로드 가능

📱 QR 코드 기능

QR 코드로 룸 공유하기

모바일 기기에서 빠르게 접속할 수 있도록 QR 코드를 생성하세요:

  1. 룸 화면에서 "QR 코드" 버튼 클릭
  2. 생성된 QR 코드를 모바일 카메라로 스캔
  3. 자동으로 해당 룸으로 이동합니다
💡 팁: QR 코드를 이미지로 저장하여 메신저나 이메일로 공유할 수 있습니다!

QR 코드로 파일 다운로드

파일 다운로드 링크를 QR 코드로 만들어 쉽게 공유하세요:

  • 파일 카드의 "QR 코드" 아이콘 클릭
  • 모바일에서 QR 스캔 → 즉시 파일 다운로드 페이지로 이동
  • 여러 파일을 선택하여 하나의 QR 코드로 공유 가능

🔒 보안 및 개인정보

파일 보관 기간

업로드된 파일은 일정 시간 후 자동으로 삭제됩니다:

  • 파일은 클라우드 스토리지에 임시 저장됩니다
  • 룸이 비활성화되면 관련 파일도 삭제됩니다
  • 민감한 파일은 공유 후 즉시 삭제하는 것을 권장합니다

안전한 사용을 위한 권장사항

⚠️ 주의:
  • 개인정보나 민감한 문서는 공유를 피하세요
  • 룸 코드를 신뢰할 수 있는 사람에게만 공유하세요
  • 파일 공유 후에는 룸을 나가거나 파일을 삭제하세요

💡 유용한 팁

클립보드 활용

스크린샷이나 복사한 이미지를 Ctrl+V (Cmd+V)로 즉시 공유할 수 있습니다!

빠른 룸 입장

URL에 직접 룸 코드를 입력하여 접속할 수 있습니다:
https://clipboardapp.org/#/123456

다중 파일 업로드

여러 파일을 한 번에 선택하거나 드래그 앤 드롭으로 업로드하세요.

실시간 동기화

같은 룸에 있는 모든 사용자는 파일 업로드/삭제를 실시간으로 확인할 수 있습니다.

❓ 자주 묻는 질문

Q: 회원가입이 필요한가요?

A: 아니요, 회원가입 없이 바로 사용할 수 있습니다!

Q: 파일 크기 제한이 있나요?

A: 일반적인 파일 크기는 지원되지만, 매우 큰 파일(수백 MB 이상)은 업로드 시간이 길어질 수 있습니다.

Q: 어떤 파일 형식을 지원하나요?

A: 이미지, 문서, 동영상, 압축 파일 등 모든 형식의 파일을 공유할 수 있습니다.

Q: 모바일에서도 사용할 수 있나요?

A: 네! 모바일 웹 브라우저에서 완벽하게 작동합니다.

Q: 파일이 다운로드되지 않아요

A: 브라우저의 팝업 차단을 해제하거나, 다른 브라우저를 시도해보세요.

더 궁금한 사항은 FAQ 페이지를 확인하세요.

🎯 사용 시나리오

📸 스크린샷 빠르게 공유하기

  1. Windows: Win + Shift + S 또는 Mac: Cmd + Shift + 4로 스크린샷 캡처
  2. Clipboard Share 룸 화면에서 Ctrl+V (Cmd+V) 붙여넣기
  3. 상대방에게 즉시 이미지가 공유됩니다!

📂 협업 파일 교환하기

  1. 팀원들과 동일한 룸 코드 공유
  2. 각자 필요한 파일을 업로드
  3. 모든 팀원이 실시간으로 파일 확인 및 다운로드
  4. 작업 완료 후 파일 삭제 또는 룸 나가기

📱 PC ↔ 모바일 파일 전송

  1. PC에서 룸 생성 및 QR 코드 표시
  2. 모바일로 QR 코드 스캔하여 입장
  3. 양방향으로 파일 업로드/다운로드 가능
